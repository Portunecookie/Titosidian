## Animated Widget
- AnimatedPositioned
- AnimatedScale -> 커지게 하는 효과
- AnimatedOpacity
- AnimatedAlign
- AnimatedContainer
- AnimatedSlide

### 장단점
- 장점
	- 사용이 간편
	- 직관적
- 단점
	- Animation level 별 디테일 하게 조절 불가
	- 무한 반복 / 실행/ 뒤로 재생 등의 기능 불가능

## Animation Controler
- 지정해서 사용 가능

### 장단점
- 장점
	- Animation level 별 디테일 하게 조절 가능
	- 무한 반복 / 실행/ 뒤로 재생 등의 기능 가능
- 단점
	- 사용을 위해 진입장벽이 높다
	- 소스코드가 복잡하다
